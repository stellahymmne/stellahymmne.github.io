Magical Language

If you have seen something which requires an act of magic, in its entirety, no you haven't. I am being perfectly literal. Paths go backward. If something is called, it means a path was found back to it. If it finds a path back to nothing, nothing goes back from nothing, it has no predecessor. Nothing is called.

If I have seen something which depends on nothing, those cannot be called forth, because nothing exists as a prerequisite substance. They have no binding parameters, and thus no constructor.

I suspect this has something to do with fixed points. If I have seen something which depends on infinity, it is uniquely defined up to isomorphism? Because there's really only one way that happens and it's like, a cast from the infinite loop?

I notice that I am confused. I learned this from the Rationalists. They seem very confused.

There is a tendency I have. It is something like path-synesthesia. It expresses the path-synesthesia in these symbols.

An object is inserted as an argument and then used once. This ensures it is the same object with logical certainty, because there could only be one path from the beginning to the end.

The selection of the first and second object from alternatives. The way such objects are arranged is with one resource and two paths forward. The path is taken when this selection of the alternative is reached. This is where the choice happens, when one option is replaced with a blank. The thing one passes in is two alternatives.

Nothing contains no objects except those generated by ambiguity. With the witness rule, identity cannot exist for it. We certainly cannot provide a witness to the void type, so it contains no objects and never will. Except for those... generated by ambiguity.
